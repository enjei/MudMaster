SendAlias


Format: /sendalias {chat name} {reference number}
Format: /sendalias {chat name} {alias name}
Format: /sendalias {chat reference number} {reference number}
Format: /sendalias {chat reference number} {alias name}

SendAlias sends one of your aliases to another chat user.

  • {chat name} The chat name of the person you want to send to.
  • {chat reference number} The number of the person you want to send to.
  • {reference number} The number of the alias you want to send.
  • {alias name} The alias you want to send.
