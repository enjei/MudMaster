RequestConnects


Format: /requestconnects {reference number}
Format: /requestconnects {chat name}

RequestConnects will return all the public connections of another chat user.
When the connections are returned Mud Master will call them all and attempt to
connect you to them. This is useful for joining a group of people on chat
without having to individually call each of the people.

  • {reference number} The number of the chat connection you want to request
         connections from.
  • {chat name} The name of the chat connection you want to request
         connections from.
