ListCopy


Format: /listcopy {list name}{new list name}
Format: /listcopy {list name}

Duplicates a user defined list and all items in the list into a new list.

  • {list name} The name of the list you want to copy.
  • {new list name} The name of the list you want to be the new copy. If no new list name is provided the new list will be called the original name with Copy appended to it.
    /listcopy {MyList} would result in two identical lists, one called MyList and the other called MyListCopy
