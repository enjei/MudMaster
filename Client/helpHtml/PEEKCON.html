PeekConnections


Format: /peekconnections {chat name}
Format: /peekconnections {chat number}

  • {chat name} - Name of the chat connection you wish to peek at.
  • {chat number} - Number of the chat connection you wish to peek at.

    This command will return you a list of connections another chat user has. This
    is very similar to request connections except it will not try to connect you
    to all of the returned connections.
