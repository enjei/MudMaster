FileStatus


Format: /filestatus {reference number}
Format: /filestatus {chat name}

FileStatus shows you the progress of a current file transfer. The name of the
file, the file size and the bytes transferred are shown.

  • {reference number} The number of the chat connection where the file
         transfer is in progress.
  • {chat name} The name of the chat connection where the file transfer is in
         progress.
