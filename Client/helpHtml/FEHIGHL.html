FEditHighlight


Format: /fedithighlight {reference number}
Format: /fedithighlight {mask}

FEditHighlight allows you to easily edit a highlight in the full screen
editor. You can either use the reference number shown when you list your
defined highlights or the mask to edit it.

  • {reference number} The number of the highlight you want to edit.
  • {mask} The mask text of the highlight you want to edit.

    NOTE: Highlights are stored and looked up by the mask. If you edit a
    highlight and you change the mask, when you press enter a new highlight
    will be created. This will not erase the highlight that you had originally
    edited. If you wish to remove the original version you need to use
    /unhighlight.
