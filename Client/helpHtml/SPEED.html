Speed


Running Mud Master from the Windows 95 Explorer will cause it to run slow.
You will be much better off if you go to a DOS shell and run the program from
there. Or make a DOS shell icon for Mud Master and play around with the
properties to give it more system time. You can turn the idle sensitivity all
the way down to LOW to give it a little more time.

Mud Master is a console mode application. Which means it looks like a DOS app
but isn't really. You will find that running it in text mode is much faster
than running it in a window. If it is running in a window you can press
[ALT]+[ENTER] to switch to text mode.
