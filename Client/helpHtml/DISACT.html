DisableAction

Format:
/disableaction {reference number}

or

/disableaction {trigger}

Description:

Disables an action, so that it no longer executes when the triggering text is seen.
  • {reference number} The number of the action to disable.
  • {text pattern} The trigger text of the action to disable.
Examples:
/disableaction 3
This will disable the action that is numbered 3 in the action list.
/disableaction {^You are %0 miles away}
This will disable the action that contains the provided trigger.
Related Commands: /action, /enableaction, /groupaction, /sendaction, /unaction
