SendList


Format: /sendlist {chat name} {reference number}
Format: /sendlist {chat name} {list name}
Format: /sendlist {chat reference number} {reference number}
Format: /sendlist {chat reference number} {list name}

SendList sends one of your lists and all the items in the list to another
chat user.

  • {chat name} The chat name of the person you want to send to.
  • {chat reference number} The number of the person you want to send to.
  • {reference number} The number of the list you want to send.
  • {list name} The list you want to send.
