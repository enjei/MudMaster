SendHighlight


Format: /sendhighlight {chat name} {reference number}
Format: /sendhighlight {chat name} {highlight text}
Format: /sendhighlight {chat reference number} {reference number}
Format: /sendhighlight {chat reference number} {highlight text}

SendHighlight sends one of your highlights to another chat user.

  • {chat name} The chat name of the person you want to send to.
  • {chat reference number} The number of the person you want to send to.
  • {reference number} The number of the highlight you want to send.
  • {highlight text} The text of the highlight you want to send.
