Session


Format: /session {session name} {network address} {port}

This is the command you use to create a connection to the mud.

  • {session name} The name of the session you want to start. This is not use
         anywhere at the moment and is just included to be compatible with tintin.
         It is still a required parameter.
  • {netword address} The address of the mud. This can be in either IP
         formation, or a named address.
  •   {port} The port you want to connect to.

    Example:
       /sesion med medievia.com 4000
