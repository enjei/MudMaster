OpenPort


Format: /openport {com port} {speed} {data bits} {parity} {stop bits}

Openport takes control of the serial port for mud master's use. Openport will
be unable to take control of the serial port if any other programs in windows
are currently using it.

   Parameter     Possible Values
   ---------     ---------------
   com port      com1, com2, com3, com4
   speed         1200, 2400, 4800, 9600, 14400, 19200, 38400, 57600, 115200
   data bits     7, 8
   parity        None, Even, Odd, Mark, Space
   stop bits     0, 1

/openport com1 57600 8 None 1
