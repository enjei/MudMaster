SendVariable


Format: /sendvariable {chat name} {reference number}
Format: /sendvariable {chat name} {variable name}
Format: /sendvariable {chat reference number} {reference number}
Format: /sendvariable {chat reference number} {variable name}

SendVariable sends one of your variables to another chat user.

  • {chat name} The chat name of the person you want to send to.
  • {chat reference number} The number of the person you want to send to.
  • {reference number} The number of the variable you want to send.
  • {variable name} The name of the variable you want to send.
