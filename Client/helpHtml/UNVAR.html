UnVariable


Format: /unvariable {reference number}
Format: /unvariable {variable name}

Removes a variable from your list of defined variables. You can either type
the number of the variable which you see when you list the variables or an
exact text math of the variable name.

  • {reference number} The number of the variable you want to remove.
  • {variable name} The name of the variable you want to remove.
