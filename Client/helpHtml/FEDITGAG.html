FEditGag


Format: /feditgag {reference number}
Format: /feditgag {mask}

FEditGag allows you to easily edit a gag in the full screen editor. You can
either use the reference number shown when you list your defined gags or the
mask to edit it.

  • {reference number} The number of the gag you want to edit.
  • {mask} The mask text of the gag you want to edit.

    NOTE: Gags are stored and looked up by the mask. If you edit a gag and you
    change the mask, when you press enter a new gag will be created. This will not
    erase the gag that you had originally edited. If you wish to remove the
    original version you need to use /ungag.
