SendEvent


Format: /sendevent {chat name} {reference number}
Format: /sendevent {chat name} {event name}
Format: /sendevent {chat reference number} {reference number}
Format: /sendevent {chat reference number} {event name}

SendEvent sends one of your events to another chat user.

  • {chat name} The chat name of the person you want to send to.
  • {chat reference number} The number of the person you want to send to.
  • {reference number} The number of the bar item you want to send.
  • {event name} The name of the event you want to send.
