Undo


Format: /undo
Format: /undo {number}

  • {number} - Number of the command in the list to undo.

    Anytime a command (alias, action, macro, etc..) is delete it is place in the
    undo list. The undo list remembers the last 50 commands deleted. /Undo without
    a number will show you a list of the most recent deletions. To undelete an
    item type /undo with the number of the item you want to restore.
