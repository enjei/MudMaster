SendArray


Format: /sendarray {chat name} {reference number}
Format: /sendarray {chat name} {array name}
Format: /sendarray {chat reference number} {reference number}
Format: /sendarray {chat reference number} {array name}

SendArray sends one of your arrays and all the items in the array to another
 chat user.

  • {chat name} The chat name of the person you want to send to.
  • {chat reference number} The number of the person you want to send to.
  • {reference number} The number of the array you want to send.
  • {array name} The array you want to send.
