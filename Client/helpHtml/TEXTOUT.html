TextOut


Format: /textout {filename}{text}{append indicator}

Sends text to a textfile.

  • {filename} The name of the file to write to.
  • {text} The text to output to the file.
  • {append indicator} Optional argument. If present text will be apppended to the end of the file.
    Otherwise text will replace anything in the file if it exists.
